## Install

```
npm install -g play_brain-games_or
```

## Start

Choose a game by entering its name in CLI:

1. `brain-calc`
2. `brain-balance`
3. `brain-even`
4. `brain-gcd`
5. `brain-prime`
6. `brain-progression`
